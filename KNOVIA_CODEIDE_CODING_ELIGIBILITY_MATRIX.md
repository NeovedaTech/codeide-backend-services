# Knovia <> CODE-IDE Coding Eligibility Matrix

## Purpose

This document lists the exact Knovia skill-role-level mappings that should launch the CODE-IDE coding environment.

All other skill validations must continue through Knovia's own assessment engine unless this matrix is updated.

## Routing Rule

Knovia backend should route to CODE-IDE only when this tuple is present in the matrix:

```text
skillId + roleId + level
```

Do not route based only on skill name or proficiency level.
